# AI詳細分析 分析方針

このファイルは、「注目記事」に選ばれた記事1件ずつに対して行う**詳細分析**の
分析方針(視点・トーン・重視する観点)を定義します。次回の収集から反映されます。
コードの変更は不要です。

下にある2行のコメント行（START/ENDのマーカー）の**間**に書いた内容だけが実際に
AIへ渡されます。マーカー行自体は削除しないでください。

出力される項目(3行要約・重要ポイント・キーワード・カテゴリ・重要度・コンサル視点・
イシュー・仮説・ビジネスへの影響・アクション・読書ノート)の構成自体は、AIが複雑な
Markdown見出しフォーマットの指示に安定して従いきれないという問題を回避するため、
コード側(js/app.js)で固定的に組み立てています。ここで自由に変えられるのは、
各項目の「中身」をどんな視点で書かせるかという分析方針の部分です。

法的リスク・規制動向・レピュテーションリスクの観点を重視して分析してください。
